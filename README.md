# House Price Prediction Using Machine Learning

This project predicts house prices using a machine learning model based on various features such as area income, house age, number of rooms, bedrooms, and population.

## Technologies Used
- Python
- Pandas
- NumPy
- Seaborn
- Matplotlib
- Scikit-learn

## Dataset Features
- Avg. Area Income
- Avg. Area House Age
- Avg. Area Number of Rooms
- Avg. Area Number of Bedrooms
- Area Population
- Price

## Project Files
- `house_price_prediction.py` : Machine learning model code
- `House_dataset.csv` : Dataset used for training and testing
- `requirements.txt` : Required Python libraries
- `README.md` : Project documentation

## Output
The model predicts house prices using a Linear Regression model and evaluates performance using MAE, MSE, and RMSE metrics.

## Model Evaluation

The Linear Regression model was evaluated using the following metrics:

- **MAE (Mean Absolute Error)** – Measures the average absolute difference between predicted and actual prices.
- **MSE (Mean Squared Error)** – Measures the average squared difference between predicted and actual values.
- **RMSE (Root Mean Squared Error)** – Square root of MSE, showing prediction error in the same units as the target variable.
